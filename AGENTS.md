# Repository Instructions for Coding Agents

Read these files before making Max-related changes:

- `max/docs/README.md`
- `max/docs/SYSTEM_ARCHITECTURE.md`
- `max/docs/TEAMMATE_CONTRIBUTIONS.md`
- `max/docs/INTEGRATION_PLAN.md`
- `max/docs/CODEX_TASK_BRIEF.md`

Do not edit files under `max/teammate_sources/`. Treat them as immutable source references.

For multi-file architectural changes, first produce a written plan and wait for approval. Keep the first stable integration separate from optional database-backed processed-star audio and website re-rendering.

Never commit credentials, downloaded user recordings, generated audio, or unreviewed private exports.
