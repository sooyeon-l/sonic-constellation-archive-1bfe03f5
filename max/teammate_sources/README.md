# Teammate Source Files

Each teammate folder should contain:

```text
<name>/
├── NOTES.md
└── original/
    └── untouched source files
```

Rules:

1. Never modify files inside `original/`.
2. Put integration-ready adaptations in `max/modules/`.
3. Keep public-repository notes sanitized: no private Notion URLs, personal contact information, or unnecessary screenshots.
4. Large videos and test audio should stay out of Git unless Git LFS is intentionally configured.
